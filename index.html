MRI Pelvic Phantom for Adaptive Radiotherapy

ECEN 403 Senior Design Project · Texas A&M University at Qatar

Team #1 2025–2026
Project summary: This project aims to design, build, and validate an MRI-compatible pelvic phantom to support adaptive radiotherapy workflows. The phantom will be used to study geometric distortion, image quality, and motion, and will be paired with a software pipeline for distortion analysis and correction.

Last updated: November 19, 2025

Project Overview

Objective

The objective of this project is to develop a realistic pelvic phantom that can be scanned on MRI systems and used to evaluate and improve adaptive radiotherapy workflows.

The phantom will emulate key pelvic structures and allow controlled motion and distortion experiments, enabling quantitative evaluation of image quality, registration accuracy, and dose calculation.

Motivation

  • Non-uniform magnetic fields in MRI can cause spatial distortions.
  • Adaptive radiotherapy relies heavily on accurate MR imaging for planning and verification.
  • A standardized pelvic phantom helps benchmark MRI performance and test correction methods.

Key Features

  • MRI-compatible pelvic phantom geometry.
  • Embedded structures representing organs and soft tissue.
  • Capability to introduce controlled motion and deformation.
  • Imaging pipeline for distortion quantification and correction.

System Design

Hardware & Phantom Design

  • Pelvic phantom geometry based on anatomical references.
  • 3D-printed components with MRI-compatible materials.
  • Internal structures representing bone, soft tissue, and target volumes.
  • Motion module to simulate patient movement or organ motion (if applicable).

Software & Image Processing

  • Acquisition of MR images under controlled conditions.
  • Quantification of geometric distortion using known phantom geometry.
  • Implementation and testing of distortion correction algorithms.
  • Comparison of pre- and post-correction images using quantitative metrics.

Deliverables

  • Functional MRI pelvic phantom.
  • Documentation of design, materials, and fabrication steps.
  • Image processing scripts / software for distortion analysis and correction.
  • Interim and final presentations and written reports.
